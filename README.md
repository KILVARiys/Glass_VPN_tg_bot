# 🛡️ Glass VPN Bot

**Telegram-бот для продажи и управления VPN-подписками на базе 3x-ui**

---

## 📋 О проекте

**Glass VPN Bot** — полнофункциональный Telegram-бот для автоматизации продажи VPN-подписок. Бот интегрируется с панелью управления **3x-ui** и платёжной системой **Platega.io**, позволяя полностью автоматизировать процесс покупки, активации и управления подписками.

### ✨ Возможности

| Функция | Описание |
|---|---|
| 🎁 Пробный период | Бесплатная пробная подписка на N дней (один раз) |
| 💳 Оплата СБП/картой | Интеграция с Platega.io для приёма платежей |
| 🔑 Промокоды | Создание и активация промокодов с бонусными днями |
| 👥 Реферальная система | Бонусные дни за приглашённых друзей |
| 🔗 Авто-выдача ключей | Автоматическое создание клиента в 3x-ui и генерация SUB-ссылки |
| 📊 Личный кабинет | Статус подписки, дата окончания, статистика рефералов |
| 🔐 Админ-панель | Управление пользователями, рассылки, промокоды, платежи |

## 🗂️ Структура проекта

```
Glass_VPN_tg_bot/
├── main.py               # Точка входа, запуск бота
├── handlers.py           # Все обработчики команд и callback-кнопок
├── keyboards.py          # Inline-клавиатуры
├── database.py           # Работа с SQLite базой данных
├── utils.py              # Клиент для 3x-ui (XUIClient)
├── payment_platega.py    # Клиент платёжной системы Platega.io
├── webhook_server.py     # Webhook-сервер для приёма уведомлений
├── config.py             # Конфигурация и переменные окружения
├── .env                  # Секретные переменные (не коммитить!)
└── requirements.txt      # Зависимости
```

## ⚙️ Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/KILVARiys/Glass_VPN_tg_bot.git
cd Glass_VPN_tg_bot
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Настройка `.env`

Создайте файл `.env` в корне проекта:

```env
# Telegram
BOT_TOKEN=your_bot_token_here

# 3x-ui панель
XUI_HOST=https://your-server.com:2053
XUI_USERNAME=admin
XUI_PASSWORD=your_password
XUI_INBOUND_ID=1

# Платежи (Platega.io)
PLATEGA_API_KEY=your_platega_api_key

# Администратор
ADMIN_ID=123456789
```

### 4. Запуск

```bash
python main.py
```

## 🔧 Конфигурация

Основные параметры в `config.py`:

| Параметр | По умолчанию | Описание |
|---|---|---|
| `SUB_DAYS` | `30` | Дней подписки при покупке |
| `TRIAL_DAYS` | `3` | Дней пробного периода |
| `PRICE_RUB` | `100` | Стоимость подписки в рублях |
| `REFERRAL_BONUS_DAYS` | `7` | Бонус за реферала в днях |

## 🤖 Команды бота

| Команда | Описание |
|---|---|
| `/start` | Главное меню |
| `/admin` | Административная панель (только для админов) |
| `/cancel` | Отмена текущего действия |

---

## 🔐 Администрирование

Войдите в панель через `/admin`. Доступны следующие функции:

- 📋 **Список пользователей** — просмотр всех пользователей со статусом подписки
- 📨 **Рассылка** — отправка сообщения всем пользователям
- ➕ **Создать промокод** — новый промокод с указанием бонусных дней и лимита использований
- 🗑️ **Удалить промокод** — удаление существующего промокода
- 💰 **Платежи** — история последних транзакций
- 👤 **Управление админами** — добавление и удаление администраторов

## 🏗️ Технологии

- **[aiogram 3.x](https://aiogram.dev/)** — асинхронный фреймворк для Telegram Bot API
- **[3x-ui](https://github.com/MHSanaei/3x-ui)** — панель управления XRay/VPN-сервером
- **[Platega.io](https://platega.io/)** — платёжный агрегатор (СБП, карты)
- **SQLite** — локальная база данных пользователей, платежей и промокодов
- **aiohttp** — асинхронные HTTP-запросы к 3x-ui API
